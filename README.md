# AI智能政企瞭望舆情收集分析系统


## 后台管理

账号登陆信息管理；


## 爬虫管理

该模块主要用于新增和管理爬虫源，可以自定义建爬虫，dist/baidusearch目录需要实现一个针对百度搜索引擎的爬虫模块，可以直接使用，可以基于该模块代码，实现数据库的管理。


## 采集管理

1、该模块主要用于用户输入关键字实现定向采集数据的功能模块，支持用户选择爬虫源（数据来源于爬虫管理模块），通过开关模式进行切换选择。
2、开始采集后，通过橱窗列表模式显示采集到数据（封面图片【如果有图片则实现，没有图片则显示默认图片】、标题【点击标题可以新窗口打开原网页】，时间、来源），用户可以选择一个或多个数据进行保存，支持将选中的数据保存到数据库中。
3、爬虫过程中，如果有数据，则即时渲染到页面，页面随数据滚动到最新位買。
4、页面布局调整为从上到下分别排列为：关键字输入框、爬虫源选择开关面板、橱窗数据列表。
5、以上未设计或完整的功能操作，自行设计完善。


## 数据管理

1、该模块为采集保存后的数据，需要通过数据库将数据检索到内面。
2、支持在页面中模糊查询数据
3、支持分页功能
4、在列表的最后一列预留操作项功能，支持：删除、AI深度采集（后续再给你详细提示完成功能开发）
5、在搜索区下方新增加一行功能区，功能有：批量删除、批量AI深度采集



## AI模型管理

1、该模块为大模型服务引擎，通过以下基本信息实现模型接入。
2、该模块将支持后续实现AI深度采集提供支持服务（后续再给你详细提示完善功能对接开发）。
3、实现模型测试，可以弹窗打开聊天对话界面，以测试模型服务和连接状态。
4、模型服务通过openai模块对接实现，API范式需要支持openai-api范式。
5、模型管理支持url.api-key，系统提示词的动态配置和管理，相关信息需要存储到数据库中。
6、需要统计AI大模型的token消耗，记录并汇总总token数据。
